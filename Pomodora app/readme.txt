Pomodoro Timer with Tkinter and Win32com
A simple timer application written in Python that implements the Pomodoro Technique. The app features a GUI built with Tkinter and includes a text-to-speech output and a beep sound to notify the user when the timer is up.

Prerequisites
To run this application, you will need to have the following software installed on your machine:

Python 3
Tkinter
Pywin32
Installing
To install the required libraries, run the following command in your terminal:

Copy code
pip install tkinter pywin32
Running the Application
To run the application, simply run the following command in your terminal:

Copy code
python pomodoro_timer.py
Usage
Enter the desired time in minutes in the text field.
Click the "Start" button to start the timer.
When the timer is up, the display will show "Time's up!", a beep will sound, and a text-to-speech output will be played.
Click the "Stop" button to stop the timer before it's up.
Click the "Reset" button to reset the timer to its original state.
Built With
Python 3 - The programming language used
Tkinter - The GUI library used
Pywin32 - The library used to play the beep sound and text-to-speech output.
